Weiss Parameter Method

This tool helps understand crystal structures by defining crystal faces based on their intercepts with crystallographic axes. It uses Weiss Parameters (relative axial intercepts) and then converts them to Miller Indices (hkl or hkil) for a standardized notation. It also explains how axial ratios, the relative lengths of these axes, are determined.

Lattice Plane Visualizer

This tool visually constructs crystallographic planes within a unit cell based on their Miller Indices (e.g., (hkl)). It involves selecting an appropriate origin (shifting if indices are negative), taking reciprocals of the indices (a zero index means the plane is parallel to that axis), and then plotting these points to draw the plane. For more details, see: YouTube Guide
